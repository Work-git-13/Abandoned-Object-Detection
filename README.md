Второе тестовое задание:
Система детекции оставленных предметов (OpenCV).

Вычитание фона: Использование алгоритма MOG2.
Фильтрация людей через HOG Descriptor: Если объект неподвижен, он проверяется детектором людей. Если в этой зоне найден человек - тревога не поднимается.

Реализация ALARM системы: Bounding Box обекта, если он остается неподвижным дольше заданного времени и не распознается модулем, как человек.
